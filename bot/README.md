## Floradetails demo Telegram-бот

Демонстрационный Telegram-бот для интернет-магазина floraldetails.ru (цветы и декор).  
Бот показывает ключевой функционал будущего продукта на тестовых данных без реальных интеграций.

### Запуск в 5 шагов

1. **Перейдите в каталог бота**:
   ```bash
   cd Detali_Bot/bot
   ```
2. **Создайте файл `.env` на основе примера**:
   ```bash
   cp .env.example .env
   ```
   Впишите ваш `BOT_TOKEN` Telegram-бота и список `ADMIN_IDS` (через запятую).
3. **Создайте и активируйте виртуальное окружение (Python 3.11+)**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
4. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
5. **Запустите бота**:
   ```bash
   python -m bot.main
   ```

После старта откройте бота в Telegram, отправьте команду `/start` и пользуйтесь главным меню:
каталог 🌸, корзина 🛒, личный кабинет 👤 и админ-панель ⚙️ (для пользователей из `ADMIN_IDS`).
