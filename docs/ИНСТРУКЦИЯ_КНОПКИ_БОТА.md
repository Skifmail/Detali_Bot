# Инструкция: кнопки бота (админ-панель)

После нажатия **Старт** / **Главное меню** под сообщением появляется клавиатура из пяти кнопок.

---

## Главное меню (нижняя клавиатура)

| Кнопка | Действие |
|--------|----------|
| **🌸 Каталог** | Открывает каталог товаров (категории, затем товары). |
| **📦 Заказы** | Переход в раздел управления заказами. |
| **📊 Статистика** | Показывает сводку: пользователи, заказы, выручка, топ товаров, выручка по городам, заказы по статусам. |
| **📣 Рассылка** | Запуск рассылки сообщения всем пользователям бота. |
| **Ещё** | Дополнительные действия: каталог, пользователи, администраторы, контакты. |

---

## 🌸 Каталог

- Появляется **список категорий** (кнопки под сообщением).
- Нажатие на категорию → **список товаров** с пагинацией (◀️ Стр. 1/2 ▶️, «← К категориям»).
- Нажатие на товар → карточка товара с кнопками **«Подробнее»**, **«🛒 В корзину»**.
- **«← К категориям»** — вернуться к списку категорий.
- **«🏠 Главное меню»** — вернуться в главное меню (появляется внизу экрана).

---

## 📦 Заказы

**Нижняя клавиатура:**

| Кнопка | Действие |
|--------|----------|
| **🔍 Найти заказ** | Запрос номера заказа (4 цифры) или телефона; показ найденных заказов. |
| **Новые** | Список заказов со статусом «Новый». |
| **В доставке** | Список заказов со статусом «В доставке». |
| **Оплаченные** | Список заказов со статусом «Оплачен». |
| **📥 Выгрузить заказы** | Выбор периода (Сегодня / Неделя / Месяц) и отправка CSV-файла с заказами. |
| **← Назад** | Возврат в главное меню. |

**Под сообщением со списком заказов** — кнопки по одному заказу (номер, статус, дата). Нажатие на заказ открывает **карточку заказа**.

**В карточке заказа:**

| Кнопка | Действие |
|--------|----------|
| **🔄 Сменить статус** | Выбор нового статуса заказа (Новый, Ожидает оплаты, Оплачен, В доставке, Доставлен, Отменён). |
| **✉️ Написать клиенту** | Ввод текста сообщения; оно отправляется клиенту в Telegram с указанием заказа и контактом для связи. |
| **❌ Отменить заказ** | Отмена заказа (кнопка не показывается, если заказ уже отменён). |
| **← Назад** | Возврат к списку заказов. |

В списке заказов при большом количестве: кнопки **◀️** и **▶️** — переключение страниц. **← Назад** — в меню заказов (нижняя клавиатура остаётся).

---

## 📊 Статистика

Показывается один текст со сводкой (пользователи, заказы, выручка, топ товаров, выручка по городам, заказы по статусам). Отдельных кнопок в этом экране нет; для возврата используйте нижнюю клавиатуру или команду /start.

---

## 📣 Рассылка

- Бот просит отправить сообщение для рассылки (текст или фото с подписью).
- Появляются кнопки **«✅ Отправить рассылку»** и **«❌ Отмена»**.
- **Отправить рассылку** — сообщение уходит всем пользователям бота.
- **Отмена** — отказ от рассылки.

---

## Ещё

**Нижняя клавиатура:**

| Кнопка | Действие |
|--------|----------|
| **🔄 Обновить каталог** | Синхронизация каталога с сайтом OpenCart (товары и категории подтягиваются заново). |
| **👥 Пользователи** | Список пользователей бота (id, Telegram ID, имя, телефон). |
| **👤 Администраторы** | Список администраторов. У главного админа (из .env) есть кнопки «➕ Добавить админа» и «➖ Удалить админа»; у остальных админов — только просмотр списка и «Назад». |
| **📞 Контакты админов для клиентов** | Редактирование контактов (до 5), которые видят клиенты при обращении в поддержку. |
| **← Назад** | Возврат в главное меню. |

**Администраторы (под сообщением):**

- **➕ Добавить админа** — ввод Telegram ID пользователя; он получает те же права, что и вы (кроме управления списком админов, если вы не главный админ).
- **➖ Удалить админа** — список админов, добавленных через бота; кнопки «❌ Удалить &lt;ID&gt;» по каждому. Главных админов (из .env) удалить через бота нельзя.
- **← Назад** — возврат в меню «Ещё».

---

## Возврат в главное меню

- В нижней клавиатуре: кнопка **«← Назад»** (в разделах Заказы и Ещё).
- Или отправить команду **/start** или нажать **«🏠 Главное меню»**, если эта кнопка отображается.
